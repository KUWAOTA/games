# ゲームプランニングシート：競争社会からの脱出！

## 1. ビジュアル・スタイル
- **グラフィック**: 低彩度のドット絵。
- **ライティング**: 主人公の部屋は常に薄暗く、PCのモニターの光やスマートフォンの青白い光が強調される。
- **UI**: 
    - 画面上部または横に、常に4つのバー（メンタル、焦り、社会への不満、プライド）を表示。
    - 古いブラウザやSNS風のウィンドウデザイン。

## 2. ゲームサイクル (Game Loop)
1. **起床**: 部屋の真ん中からスタート。
2. **行動フェーズ**: 
    - 制限時間または「行動ポイント」を消費。
    - アイテム（本棚、PC、冷蔵庫など）を選択して各アクションを実行。
3. **Twitterフェーズ（帰宅/終了演出）**: 
    - ブラックアウト後、スマホ画面が表示。
    - その日のパラメータに合致した「病みツイート」がタイムラインに流れる。
4. **睡眠/翌日**: 
    - ベッドへ行くことで日付更新（メンタル管理上、ベッドに行かない選択も可能）。

## 4. Twitter(X)演出の詳細
- **バリエーション**: Discordから取得した「メンタルマシ」「ちょい悪」「悪るる」「社会への不満」「プライド」の5カテゴリに基づき、各5〜10パターン程度用意。
- **演出**: タイピング音とともに一文字ずつ表示され、最後に「送信ボタン」が勝手に押されるアニメーション。

## 5. 未解決・要検討事項
要件定義とプランニングを作成してみて、以下の点がまだ詰める必要があると感じました：

1. **ゲームの終了期限**: 
   - 何日生き残れば「クリア」なのか？（例：30日間）
   - あるいは、どのエンディングにも到達せずに期限が来たらどうなるのか？
2. **「成功」エンドの定義**:
   - 資料の中に「社会への不満系 - 成功」や「刑務所内で虚脱 - 成功」という記述がありますが、これはプレイヤーにとっての「望ましい結末」としての成功か、あるいは「その道に振り切ったこと」を指すのか。
3. **パラメータの初期値・閾値**:
   - 各バーのMAX値と、エンディングが発生する具体的な数値条件。
4. **「勉強」の目的**:
   - 勉強し続けることで得られる「メリット」はあるのか？（例：資格取得エンドなど、隠し要素）
   - それとも、「し続けなければならない呪い」としてのフレーバー要素か。
