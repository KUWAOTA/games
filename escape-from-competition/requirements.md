# 要件定義書：競争社会からの脱出！ (Escape from Competition)

## 1. プロジェクト概要
本作は、高学歴でありながら競争社会のシステム（ラットレース）に疲弊した主人公「山下樹也」の日常と精神的葛藤を体験する、マルチエンディング形式の脱出・パラメータ管理シミュレーションゲームである。

## 2. ターゲット
- 現代社会の競争や学歴社会に対して息苦しさを感じている層
- 『NEEDY GIRL OVERDOSE』のような、重いテーマや「病み」要素を含むシミュレーションを好む層
- ドット絵やダークな雰囲気のブラウザゲーム愛好家

## 3. ゲームの核心的（コア）体験
- **閉鎖的な日常**: 薄暗い一室での単調な繰り返し。
- **不可逆な消耗**: 「メンタル」の回復手段をあえて排除し、じわじわと追い詰められていく感覚。
- **SNSによる増幅**: Twitter（X）を通じて自己の内面を吐き出し、ネット上の「強い人」との比較でさらに病んでいく現代的な悪循環。

## 4. 主要な機能要件
### 4.1. パラメータ管理システム
以下の4つのリソースを管理し、それらの値によってエンディングが分岐する。
- **メンタル**: 行動によって減少。0になると致命的な状態へ。
- **焦り**: 勉強やSNS閲覧で増加。
- **社会への不満**: 特定のアイテム使用や出来事で増加。
- **プライド**: 主人公の根本的な自己評価。エンディングの質（成功/失敗）を左右する。

### 4.2. 時間遷移システム
- 1日単位で進行。
- 1日の終わりに「Twitter画面」を経由し、翌日へ遷移する。

### 4.3. インタラクティブ・ルーム（脱出ゲーム要素）
- 部屋の中のアイテム（PC、ベッド、冷蔵庫、本棚、引き出し等）を調査し、使用することができる。
- アイテム使用によってパラメータが変動。

### 4.4. Twitterシミュレーション
- 帰宅時（1日の終わり）に、現在のパラメータに基づいた「病みツイート」が自動生成・表示される演出。
- プレイヤーの行動の「結果」としてのフィードバック。

## 5. エンディング設計
大きく分けて以下の系統を用意する：
1. **メンタル崩壊系**: 自殺、他害（会社内殺人）、緊急搬送。
2. **社会不満系**: 社会運動（反抗）、学歴の暴力（SNS依存）。
3. **その他**: 薬への依存（お薬）、社会に呑み込まれる（バッドエンド）。

## 6. 技術スタック（推奨）
- **構成**: HTML5 / CSS3 (Vanilla CSS) / JavaScript (Vanilla JS)
- **公開プラットフォーム**: GitHub Pages (既存の `KUWAOTA/games` に統合可能)
